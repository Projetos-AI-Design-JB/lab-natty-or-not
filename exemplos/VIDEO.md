# Vídeo: Uma influencer Fake Natty 🎥

## 📒 Descrição
Este vídeo é um exemplo do avanço dos modelos de vídeo IA.

## 🤖 Tecnologias Utilizadas
- IA Generativa **[Gemini 3.1](https://gemini.google.com/)** para criaçao de imagem do avatar virtual;
- IA Generativa **[Veo 3.1](https://aistudio.google.com/models/veo-3?)** geração de avatar virtual com voz;
- IA Generativa **[Nano Banana 2](https://gemini.google.com/)** para criaçao de imagem do avatar virtual;

## 🧐 Processo de Criação
Usei o "ChatGPT" para escrever e revisar o roteiro do vídeo. O "D-ID" foi essencial para criar um avatar virtual e voz sintetizada, trazendo uma dimensão futurística ao vídeo. A montagem e edição foram realizadas no Adobe Premiere para garantir qualidade profissional.

## 🚀 Resultados
O vídeo é uma cápsula do tempo que leva o espectador a uma jornada rápida pelo mundo das IAs Generativas, destacando seu potencial transformador.
